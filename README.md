# Blinkit Sales Performance Dashboard (Excel)

**Author:** Shruti Chavan  
**Tool:** Microsoft Excel (.xlsx)  
**Data:** Simulated Blinkit-style sales dataset  
**Status:** Portfolio project

## Project Summary
A professional Excel dashboard analyzing Blinkit’s sales data.  
This dashboard visualizes key business metrics such as **Total Sales, Average Rating, Outlet Type, Item Type, and Fat Content distribution** using charts, KPIs, and slicers for interactivity.  
Designed for clear, clean presentation — suitable for screen viewing and portfolio showcase.

## Files
- `Blinkit_Sales_Analysis_ShrutiChavan.xlsx` — working Excel dashboard  
- `data/blinkit_sales_dataset.csv` — sample dataset used in the project  
- `dashboard/screenshots/01_dashboard_overview.png` — dashboard preview 

## How to View
- You can preview dashboard screenshots in the `dashboard/screenshots` folder.  
- To explore the interactive dashboard, download the `.xlsx` file and open in Excel.
